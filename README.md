# Tactical Strength

La tua scheda di allenamento digitale per smartphone.

## Caratteristiche

- **5 giorni di allenamento** con esercizi dettagliati
- **Timer di recupero** automatico per ogni esercizio
- **Tracciamento** di peso e reps per ogni sessione
- **Interfaccia mobile-friendly** con React e Tailwind CSS
- **Modal dettagliati** con setup, esecuzione, respirazione e errori da evitare

## Installazione

```bash
npm install
```

## Avvio

```bash
npm start
```

L'app si aprirà su http://localhost:3000

## Struttura

- `src/data/exercises.js` - Database degli esercizi
- `src/components/ExerciseModal.js` - Modale dettagli esercizio
- `src/components/ExerciseList.js` - Lista esercizi del giorno
- `src/App.js` - Componente principale

## Deploy

L'app è pronta per essere deployata su Vercel, Netlify o qualsiasi piattaforma statica.
